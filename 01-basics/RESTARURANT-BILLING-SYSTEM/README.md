## Description
This project is a Python-based **restaurant ordering and billing system**
that allows customers to select food items from a predefined **menu**,
order multiple items, and get the **total bill amount**.

It simulates a **real-life restaurant experience** using simple Python logic.

---

## Ordering Logic
- A fixed menu with item names and prices is displayed
- User selects food items by entering item names
- User can order **multiple items**
- Total bill is calculated by **adding prices of ordered items**
- At the end, feedback is collected from the user

---

## Features
- Displays a restaurant menu with prices
- Allows multiple food item orders
- Calculates total number of items ordered
- Generates final bill amount
- Takes customer feedback
- Simple and interactive command-line experience
