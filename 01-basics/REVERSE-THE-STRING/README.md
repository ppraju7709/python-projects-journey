## Description
This project is a Python program that reverses a given string entered by
the user.

Example:
- Input: `hello`
- Output: `olleh`

The program works with words, sentences, and spaces.

---

## Features
- Reverses any string input
- Works with words and full sentences
- Simple and clear output
- Beginner-friendly logic

---

## Concepts Used
- Variables and Data Types
- User Input and Output
- Loops (`for`)
- String manipulation
- Basic logic building
