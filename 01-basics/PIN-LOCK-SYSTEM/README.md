## Description
This project is a simple Python program that simulates a **PIN-based lock system**.

The user first sets a PIN and then must enter the correct PIN to unlock
the system. If the entered PIN is incorrect, the program keeps asking
until the correct PIN is entered.

---

## Features
- User sets a PIN
- Verifies the PIN using a loop
- Keeps asking until the correct PIN is entered
- Displays success message after unlocking

---

## Concepts Used
- Variables
- User Input and Output
- While Loop
- Conditional Checking
- Basic security logic
