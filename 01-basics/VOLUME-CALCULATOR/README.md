## Description
This project is a Python program that calculates the **volume of different
3D shapes** based on user input.

The user selects a shape, enters the required dimensions, and the program
calculates the volume.

---

## Shapes Supported
- Cube
- Cone
- Sphere
- Cylinder
- Triangular Pyramid
- Rectangular Prism
- Triangular Prism
- Square Pyramid

---

## Features
- Calculates volume of multiple shapes
- Menu-driven program
- Simple and beginner-friendly logic
- Clear output

---

## Concepts Used
- Variables and Data Types
- User Input and Output
- Conditional Statements (`if-elif-else`)
- Arithmetic Operations
- Basic mathematical formulas
