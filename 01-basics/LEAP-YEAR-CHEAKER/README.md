## Description
This project is a Python program that checks whether a given year is a
**leap year** or **not**.

A leap year has **366 days** instead of 365 and occurs every 4 years,
with some special conditions.

---

## Features
- Checks whether a year is a leap year
- Uses correct leap year rules
- Simple and beginner-friendly logic
- Clear output

---

- User Input and Output

- Logical operators
