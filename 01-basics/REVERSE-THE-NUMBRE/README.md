## Description
This project is a Python program that reverses the digits of a given number.

For example:
- Input: `1234`
- Output: `4321`

The program also works correctly for **negative numbers**.

---

## Features
- Reverses digits of any integer
- Handles negative numbers
- Simple and clear output
- Beginner-friendly logic

## Concepts Used
- Variables and Data Types
- User Input and Output
- Loops (`while`)
- Arithmetic Operators (`%`, `//`)
- Basic logic building
