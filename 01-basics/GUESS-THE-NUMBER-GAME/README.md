# GUESS THE NUMBER GAME

## Description
This is a simple Python game where the computer randomly selects a number,
and the user tries to guess it correctly.

The program gives hints whether the guessed number is **too high** or **too low**
until the correct number is guessed.

---

## Concepts Used
- Variables
- User Input
- Conditional Statements (`if`, `else`)
- Loops (`while`)
- `random` module
---

## How the Game Works
1. The computer generates a random number.
2. The user enters a guess.
3. The program compares the guess with the generated number.
4. Hints are given until the user guesses correctly.
