# Multiplication Table Generator ✖️

## Description
This project is a Python program that generates the multiplication table
for a given number. The user provides a number, and the program displays
its multiplication table up to a specified range.

---

## Features
- Generates multiplication table for any number
- User-defined range (e.g., 1 to 10)
- Clear and readable output
- Simple and beginner-friendly logic

---

## Concepts Used
- Variables and Data Types
- User Input and Output
- Loops (`for` loop)
- Arithmetic Operators
- String Formatting

---

## How to Run
This project can be run locally using Python.

```bash
python main.py
