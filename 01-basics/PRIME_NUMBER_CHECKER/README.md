## Description
This project is a Python program that checks whether a given number is a
**prime number** or **not**.

A prime number is a natural number greater than 1 that has exactly two
distinct positive divisors: 1 and itself.

---

## Features
- Checks whether a number is prime or not
- Handles non-prime edge cases (0, 1)
- Clear and user-friendly output
- Efficient checking using loops

---

## Concepts Used
- Variables and Data Types
- User Input and Output
- Conditional Statements (`if-else`)
- Loops (`for`)
- Boolean logic
- Basic mathematical reasoning
