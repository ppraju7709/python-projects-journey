## Description
This project is a Python program that generates the **Fibonacci series**
up to a specified number of terms.

The Fibonacci series is a sequence where each number is the sum of the
two preceding numbers, starting from 0 and 1.

Example:
0, 1, 1, 2, 3, 5, 8, 13, ...

---

## Features
- Generates Fibonacci series for a given number of terms
- Handles basic edge cases (0 or 1 term)
- Clear and readable output
- Beginner-friendly logic
