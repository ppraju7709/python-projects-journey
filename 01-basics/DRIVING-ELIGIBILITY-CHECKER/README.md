## Description
This project is a Python program that checks whether a person is
**eligible to drive** based on their **age** and whether they have a
**valid driving license**.

---

## Eligibility Rules
- Age must be **18 years or above**
- The person must have a **valid driving license**

If both conditions are satisfied, the person is eligible to drive.

---

## Features
- Checks driving eligibility using age and license status
- Simple and beginner-friendly logic
- Clear and understandable output
