## Description
This project is a Python program that checks whether a person is
**eligible to donate blood** based on their **age** and **weight**.

---

## Eligibility Rules
- Age must be **18 years or above**
- Weight must be **45 kg or above**

If both conditions are satisfied, the person is eligible to donate blood.

---

## Features
- Checks blood donation eligibility
- Uses age and weight conditions
- Simple and beginner-friendly logic
- Clear output messages
